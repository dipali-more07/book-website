GLibrary — Discover Your Next Read

GLibrary is a frontend web application that helps users discover books across multiple genres using a fast and interactive search interface. It includes a built-in book database, category browsing, and real-time search suggestions — all running completely on the client side.

Project Overview

GLibrary is designed as a simple digital library prototype where users can quickly search for books by title, author, or genre. The interface is responsive, clean, and optimized for quick exploration and filtering of book data.


Key Features

- Search books by title, author, or genre

- Instant search results dropdown

- Structured in-browser book database

- Popular category showcase section

- Responsive design for mobile and desktop

- Modern UI with styled navigation and hero section

- Enter-key and button-based search support

- Auto-hide results when clicking outside search area


 Tech Stack

HTML5 — Structure

CSS3 — Styling & responsive layout

Vanilla JavaScript — Search logic & filtering

Client-side data model — Book & author dataset
